# Sumit Santhosh Nair - Portfolio

Personal portfolio website built with Next.js 16, Tailwind CSS, and Framer Motion.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide Icons

## Features

- IDE-inspired design with code background animation
- Responsive layout with mobile-first approach
- Git changelog style experience page
- Bento grid about section
- Magnetic navigation dock
- SEO optimized with sitemap, robots.txt, and llms.txt

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
app/
├── components/       # React components
├── experience/       # Experience page
├── hooks/           # Custom React hooks
├── lib/             # Data and utilities
├── layout.tsx       # Root layout with SEO
├── page.tsx         # Home page
└── sitemap.ts       # Dynamic sitemap
public/
├── profile.jpg      # Profile image
├── projects/        # Project screenshots
├── robots.txt       # Crawler rules
├── llms.txt         # AI crawler content
└── manifest.json    # PWA manifest
```
